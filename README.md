# Employee Data Analysis

## Overview

Employee Data Analysis is a data analysis project that aims to identify the key factors influencing employee turnover. The project uses an employee dataset from Kaggle to perform Exploratory Data Analysis (EDA), uncover patterns in employee behavior, and generate insights that can help organizations improve employee retention.

## Problem Statement

Employee data is a significant challenge for organizations as it leads to increased recruitment costs, loss of experienced employees, and reduced productivity. The objective of this project is to analyze employee data to identify the factors that contribute to employee attrition and prepare the dataset for predictive machine learning models.

## Dataset

* **Source:** Kaggle
* **Dataset Type:** Employee Dataset (CSV)
* **Target Variable:** `left`

  * `0` – Employee Stayed
  * `1` – Employee Left

## Project Workflow

1. Import required libraries.
2. Load the dataset from Kaggle.
3. Explore the dataset.
4. Perform data cleaning.
5. Check and remove duplicate records.
6. Analyze missing values.
7. Generate descriptive statistics.
8. Detect and remove outliers using the IQR method.
9. Perform Exploratory Data Analysis (EDA).
10. Visualize employee trends using graphs.
11. Analyze feature relationships using a correlation heatmap.

## Exploratory Data Analysis

The following analyses were performed:

* Dataset overview
* Missing value analysis
* Duplicate value analysis
* Descriptive statistics
* Distribution analysis
* Employee attrition distribution
* Salary analysis
* Department-wise analysis
* Promotion analysis
* Work accident analysis
* Satisfaction level analysis
* Correlation analysis

## Key Insights

* Employees with lower satisfaction levels were more likely to leave the company.
* Employees receiving lower salaries showed higher attrition rates.
* Employees who were not promoted had a greater tendency to leave.
* Employee attrition varied across different departments.
* Correlation analysis highlighted relationships between employee characteristics and attrition.
